## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/arman-danesh/my-resume.git
```

2. Navigate to the project directory
```bash
cd my-resume
```

3. Install dependencies
```bash
npm install
```

4. use build command to build typeScript & Sass
```bash
npm build
```
